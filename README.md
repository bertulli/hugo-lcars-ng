# LCARS-NG (Next Generation)

## Features
**LCARS-NG** is a Hugo theme inspired by the LCARS operating system from the Star Trek franchise (¬© CBS Television Studios), since _Star Trek: The Next Generation_ (TNG) onward. I want to clearly say it is an **unofficial** homage, not related with the propriety, intellectual or commercial, of the Star Trek franchise.

For now, it features a simple sidebar inspired by the "_okudagram_" look and feel.

## Installation

Simply clone the repository inside your Hugo project, under `<project-root>/themes/`, and point the theme in your `hugo.toml` like
```toml
theme = 'hugo-lcars-ng'
```

## Configuration
This is still a very alpha level project. For now, you can directly modify the `lcars-ng/assets/css/main.css` file.
